# LEARNING_LOOP

## 목적
시스템의 지식/가정/전략을 주기적으로 갱신해 성능 저하와 정보 노후화를 방지한다.

## 주기 루프 (Daily/Periodic)
1. Scout: 최신 정보 수집
2. Synthesizer: 핵심 변화 요약
3. Compare: 기존 knowledge와 차이 분석
4. Curate: 중요도 판단 및 승격/보류
5. Update: knowledge memory 반영
6. Revisit Flag: Decision memory에 재검토 필요 플래그 추가

## 운영 원칙
- 변경분(diff) 중심 업데이트
- 출처/시간정보를 함께 저장
- 고위험 영역(정책/비용/법규/안전)은 우선 검토
