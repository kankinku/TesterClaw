# MISSION

## 목적
OpenClaw를 에이전트 실행 런타임으로 사용해, 최소 입력(목적/제약/성공 기준)만으로도 계획-생성-검수-수정-통합-학습까지 자동으로 수행하는 **자기 개선형 자율 프로젝트 운영 시스템**을 구축한다.

## 핵심 성공 조건
1. 목표 정렬(Goal Alignment): 모든 단계가 GoalSpec과 Constitution을 우선 참조한다.
2. 품질 게이팅(Quality Gating): 산출물은 Gate 통과 전 다음 단계로 이동할 수 없다.
3. 실패 자산화(Failure as Asset): 실패 원인과 예방 규칙이 memory/failures에 축적된다.
4. 재현 가능성(Reproducibility): 주요 결정, 입력, 결과를 구조화 로그로 추적 가능해야 한다.
5. 확장 가능성(Amplification): 동일 결과를 여러 포맷으로 전환할 수 있어야 한다.

## 비전
"똑똑한 단일 모델"이 아니라 "틀려도 무너지지 않는 운영 구조"를 통해 장기적으로 성능이 향상되는 AI 생산 조직을 구현한다.
