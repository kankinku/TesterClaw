# OUTPUT_MULTIPLIER

## 목적
하나의 핵심 산출물을 다양한 채널/목적에 맞게 재구성해 가치 레버리지를 만든다.

## 예시 파이프라인
리서치 결과 1개 →
- PRD 초안
- 블로그 초안
- X 스레드
- 발표 슬라이드 아웃라인
- 데모 코드 아이디어
- 사업/투자 가설 요약

## 역할 분리
- Repurposer: 메시지/구조 재조합
- Formatter: 채널 포맷 변환
- Distributor(선택): 배포 패키지 생성

## 품질 기준
- 원문 의미 보존
- 채널별 가독성 최적화
- 목표 독자 적합성
- 과장/허위 확장 금지
